# Changelog

All notable changes to The AI Startup Game will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/), and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025

### Added
- Initial public release
- 8 AI sectors: Healthcare, Robotics, Finance, Creative AI, Cybersecurity, Climate Tech, Legal Tech, Education
- 12 founder types with unique strengths and weaknesses
- 300+ unique events and scenarios
- 3 AI-controlled rival companies
- Multiple funding paths: bootstrap, angels, accelerators, venture capital
- Strategic pivot system when competitors advance or opportunities emerge
- Built-in tutorial (5-10 minutes)
- Sound effects and audio feedback
- End-game summary with PDF export
- Mobile-responsive design (optimised for desktop)
- Offline play support via downloadable HTML file

### Documentation
- README with quick start guide and citation information
- Teaching Guide with lesson plans for various course formats
- Quick Start Guide (one-page student handout)
- FAQ for common questions and troubleshooting

## [Unreleased]

### Planned

- Additional sectors and variants

## Version History Notes

This is the first public release of The AI Startup Game. The game was developed as part of an ESRC-funded research project at Warwick Business School and Politecnico di Milano.

For questions about specific changes or to suggest features for future releases, please use the GitHub Discussions or Issues.
